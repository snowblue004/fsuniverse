# FS Birthday Museum

A static HTML, CSS, and JavaScript birthday website.

## Run locally

Open `index.html` in a browser.

For the best experience, use a simple local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Replace the placeholders

Place the real files inside:

- `assets/images`
- `assets/gifs`

Then replace the placeholder `<div>` elements in `index.html` with normal `<img>` elements.

## Notes

- The page contains no em dashes.
- Nothing has been added after the final birthday message.
- Tea is treated as something FS loves, not as a recommendation.
- The project works on GitHub Pages and Netlify.


## Background music

Place your MP3 file here:

`assets/audio/journey.mp3`

The page does not autoplay it. Visitors start and pause the music using the floating button at the top.
